# 1000 Cameras Analysis Shiny App

## Overview

This Shiny app explores a dataset containing information about 1000 cameras. Users can interactively filter and visualize the data to gain insights into camera specifications, prices, and release years. The app provides a dynamic and user-friendly interface for exploring the diverse range of cameras in the dataset.

## Shiny App Link

[Click here to access the 1000 Cameras Analysis Shiny App](https://pszy5r-jacky-yiu.shinyapps.io/1000_Camera_Analysis/)

## Dataset

The dataset used in this app is sourced from Kaggle and is titled "1000 Cameras Dataset" ([Link to Kaggle Dataset](https://www.kaggle.com/datasets/crawford/1000-cameras-dataset)). The dataset contains comprehensive information about various camera models, including details such as manufacturer, model, price, resolution, and release date.

## Features

1. **Filtering:** Users can filter cameras based on price range, manufacturer, and release year.

2. **Visualization:** A dynamic histogram visualizes the distribution of maximum resolution for the selected cameras.

3. **Result Text:** The app provides practical feedback by displaying the number of results found based on the applied filters. The result text is designed to be engaging and informative.

4. **Downloads:** Users can download the customized plot and the filtered data as a CSV file for further analysis.

## Usage

1. **Price Range:** Adjust the price range slider to filter cameras within a specific price range.

2. **Manufacturer:** Select one or more manufacturers to filter cameras produced by specific brands.

3. **Release Year:** Use the release year slider to filter cameras released within a specific time frame.

4. **Download Plot:** Click the "Download Plot" button to save the customized plot as a JPG file.

5. **Download CSV:** Click the "Download CSV" button to export the filtered data as a CSV file.
